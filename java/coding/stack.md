## Intialization

```java
Stack<T> stack = new Stack<>();

// Add element on top: push method
stack.push(object);

// Access top element without remove: peek method
stack.peek();

// Remove elements: pop method
stack.pop();

// check if stack is empty
stack.empty();

// check whether element exists in the stack. It returns the position of the element from the top of the stack. Else, it returns -1.
stack.search(object);

// there are many other methods inherited from Vector in java, as Stack extends Vector implements List extends Collection extends Iterable
```
